# משחק סדר הא-ב

אתר אינטראקטיבי קטן שבו המשתמש גורר כרטיסיות של אותיות בעברית ומסדר אותן לפי סדר האלפבית.

## מה כלול

- כרטיסיות של 22 אותיות הא-ב עם מילה תומכת לכל אות.
- גרירה ושחרור בין אזור כרטיסיות לאזור הסידור.
- בדיקת תקינות הסדר עם משוב מיידי.
- כפתור ערבוב מחדש.
- תצוגה רספונסיבית למובייל ולדסקטופ.

## הרצה מקומית

אפשר לפתוח את `index.html` ישירות בדפדפן.

או להריץ שרת מקומי:

```powershell
python -m http.server 5500
```

ואז להיכנס לכתובת:

- http://localhost:5500

## קבצים מרכזיים

- `index.html` - מבנה האפליקציה.
- `styles.css` - עיצוב ואנימציות.
- `app.js` - לוגיקת המשחק והגרירה.
